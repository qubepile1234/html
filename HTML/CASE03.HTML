▼ ゲーム操作について ▼


【タイトルメニュー】

ゲームを起動するとタイトル画面が表示されます。
各メニューの説明は以下の通りです。



最初からゲームを始めからプレイします。
続きからゲームをセーブしたところから再開します。
リストが表示されますので、続きをしたいデータを選んでください。
おまけおまけメニューを開きます。
設定　　ゲームのプレイ環境を設定します。
終了ゲームを終了します。


【おまけメニュー】

おまけを選択するとおまけ画面が表示されます。
各メニューの説明は以下の通りです。



ＣＧ鑑賞ゲーム中に見たＣＧを見ることができます。
サムネイルでリストが表示されますので、見たいＣＧを選んでください。
シーン鑑賞ゲーム中に見たシーンを再現できます。
リストが表示されますので、見たいシーンを選んでください。
ＢＧＭ鑑賞音楽を視聴します。
メインメニューに戻るタイトル画面に戻ります。


【ゲームのすすめ方】

ゲームをスタートすると、文章が次々と画面上に映し出されます。



文章の区切りに が表示されます。
「Ｅｎｔｅｒキー」「マウス左クリック」を押すと次の文章が現れます。
「マウス左ボタンのダブルクリック」でメッセージの表示スピードを速めることができます。
「Ｃｔｒｌキー」で文章を高速に読み進めることができます。
また、「Ｅｎｔｅｒキー」を押し続けることで一度読んだ文章だけを高速に読み進めることができます。

※　ご注意　※
エンディングの記録を行うため、エンディングを見終わった後で、かならず一度タイトルに戻ってからゲームを終了させてください。

【操作スイッチ一覧】

■マウスでの操作
ゲーム本編での全ての操作はマウスで行えます。

左クリックシナリオの読み進み、選択肢の決定
右クリックメッセージウィンドウの消去／表示
ホイールを上（↑）に回転メッセージ履歴の表示
ホイールを下（↓）に回転シナリオの読み進み

■キーボードでの操作
補助的に活用すれば快適なプレイができます。

Ｅｎｔｅｒシナリオの読み進み、次へ進む、選択肢の決定
Ｅｎｔｅｒ（押し続け）既読メッセージのスキップ
Ｃｔｒｌメッセージスキップ
スペースメッセージウィンドウの非表示／表示を切り替え
ＥＳＣ各種キャンセル、ゲームの終了

カーソル(↑↓→←)選択肢のカーソル移動
ＴＡＢ選択肢のカーソル順移動
ＳＨＩＦＴ＋ＴＡＢ選択肢のカーソル逆移動

ＳＨＩＦＴ＋↑メッセージ履歴の表示
ＡＬＴ＋Ｅｎｔｅｒウィンドー／フルスクリーン、モード切り替え


【ゲーム中の各種メニューについて】



■メニューボタン

[ Q.SAVE ]クイックセーブ
[ Q.LOAD ]クイックロード
[ SAVE ]セーブ画面を開きます。
[ LOAD ]ロード画面を開きます。
[ SKIP ]スキップモード。
[ AUTO ]オートモード。
[ VOICE ]音声を再生
[ LOG ]メッセージ履歴の表示
[ CONFIG ]ゲーム環境の設定
[ TITLE ]『タイトル画面に戻りますか？』確認ウインドウが表示されます。
はいを選択しますと、タイトル画面に戻ります。
[ EXIT ]『ゲームを終了しますか？』確認ウインドウが表示されます。
はいを選択しますと、ゲームが終了します。
[ × ]メッセージウインドウを閉じます。

Qキークイックセーブ
Wキークイックロード
Sキーセーブ
Lキーロード
ESCキーメッセージウインドウ消去


【メッセージ履歴画面】



回想文章は本編を読み進めている最中にマウスのローラーを↑回転させることで閲覧することができます。
画面右隅の上下にある「▲ボタン」「▼ボタン」「サイドバー」で送ったり戻したりすることができます。
音声があるキャラクターのセリフには のマークが付き、マークをクリックすることで音声を聞くことが可能です。



【プレイ環境の設定について】

設定画面でゲーム中の各種設定を行うことができます。



■環境設定

画面モードスクリーンモードを設定します。

自動改ページオートモード時の文章の表示時間を設定します。
文章スキップ文章スキップを設定します。
文字色変化特定のキャラクターの文字の色を変化するかしないかを設定出来ます。
外字表示　[ＯＮ／ＯＦＦ]外字表示を設定します。
既読文章の色既読文章の文字の色を変化するか設定します。
全体音量ボリューム全体音声の再生音量を設定します。
ＢＧＭ音量ボリューム音楽の再生音量を設定します。
ＳＥ音量ボリューム効果音の再生音量を設定します。
[テスト再生]ボタンを押して再生テストが可能です。
ＶＯＩＣＥ音量ボリュームキャラクター音声の再生音量を設定します。
[テスト再生]ボタンを押して再生テストが可能です。
キャラボタンで各キャラに切り替わります。
システム音声　[ＯＮ／ＯＦＦ]システム音声のＯＮ／ＯＦＦを設定します。
音声継続　[ＯＮ／ＯＦＦ]クリックした時に音声継続を設定します。
エフェクト機能　[ＯＮ／ＯＦＦ]エフェクト機能のＯＮ／ＯＦＦを設定します。
ツールチップ表示　[ＯＮ／ＯＦＦ]ツールチップ表示を使用するか設定します。
ムービー再生　[ＯＮ／ＯＦＦ]ムービー再生のＯＮ／ＯＦＦを設定します。
ウインドウ開閉確認　[ＯＮ／ＯＦＦ]各ウインドウ開閉確認のＯＮ／ＯＦＦを設定します。
初期化環境設定を初期値に戻します。
取り消し設定した内容を取り消して、環境設定画面を閉じます。
決定設定した内容を適用して、環境設定画面を閉じます。
また右クリックでも同等の機能が発生します。
